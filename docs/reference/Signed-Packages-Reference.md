## <a name="certificate-requirements"></a><span data-ttu-id="d4770-119">Zertifikatanforderungen</span><span class="sxs-lookup"><span data-stu-id="d4770-119">Certificate requirements</span></span>

<span data-ttu-id="d4770-120">Zum Signieren von Paketen ist ein Code Signaturzertifikat erforderlich, bei dem es sich um einen speziellen Zertifikattyp handelt, der für den `id-kp-codeSigning` Zweck [[RFC 5280 section 4.2.1.12](https://tools.ietf.org/html/rfc5280#section-4.2.1.12)] gültig ist.</span><span class="sxs-lookup"><span data-stu-id="d4770-120">Package signing requires a code signing certificate, which is a special type of certificate that is valid for the `id-kp-codeSigning` purpose [[RFC 5280 section 4.2.1.12](https://tools.ietf.org/html/rfc5280#section-4.2.1.12)].</span></span> <span data-ttu-id="d4770-121">Außerdem muss das Zertifikat eine öffentliche RSA-Schlüssellänge von 2048 Bits oder höher aufweisen.</span><span class="sxs-lookup"><span data-stu-id="d4770-121">Additionally, the certificate must have an RSA public key length of 2048 bits or higher.</span></span>

<span data-ttu-id="d4770-123">Signierte Pakete sollten einen RFC 3161-Zeitstempel enthalten, um die Gültigkeit der Signatur außerhalb der Gültigkeitsdauer des Paket Signatur Zertifikats sicherzustellen.</span><span class="sxs-lookup"><span data-stu-id="d4770-123">Signed packages should include an RFC 3161 timestamp to ensure signature validity beyond the package signing certificate's validity period.</span></span> <span data-ttu-id="d4770-124">Das zum Signieren des Zeitstempels verwendete Zertifikat muss für den `id-kp-timeStamping` Zweck [[RFC 5280 section 4.2.1.12](https://tools.ietf.org/html/rfc5280#section-4.2.1.12)] gültig sein.</span><span class="sxs-lookup"><span data-stu-id="d4770-124">The certificate used to sign the timestamp must be valid for the `id-kp-timeStamping` purpose [[RFC 5280 section 4.2.1.12](https://tools.ietf.org/html/rfc5280#section-4.2.1.12)].</span></span> <span data-ttu-id="d4770-125">Außerdem muss das Zertifikat eine öffentliche RSA-Schlüssellänge von 2048 Bits oder höher aufweisen.</span><span class="sxs-lookup"><span data-stu-id="d4770-125">Additionally, the certificate must have an RSA public key length of 2048 bits or higher.</span></span>

<span data-ttu-id="d4770-126">Weitere technische Details finden Sie in der [Paket Signatur Technical Spezifikationen](https://github.com/NuGet/Home/wiki/Package-Signatures-Technical-Details) (GitHub).</span><span class="sxs-lookup"><span data-stu-id="d4770-126">Additional technical details can be found in the [package signature technical specs](https://github.com/NuGet/Home/wiki/Package-Signatures-Technical-Details) (GitHub).</span></span>
