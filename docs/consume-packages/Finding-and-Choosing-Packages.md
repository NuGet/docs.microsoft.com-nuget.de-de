## <a name="search-syntax"></a><span data-ttu-id="ccf83-194">Suchsyntax</span><span class="sxs-lookup"><span data-stu-id="ccf83-194">Search Syntax</span></span>

<span data-ttu-id="ccf83-195">Die NuGet-Paketsuche auf nuget.org, über die NuGet-CLI und mit der Erweiterung des NuGet-Paket-Managers in Visual Studio funktionieren auf die gleiche Weise.</span><span class="sxs-lookup"><span data-stu-id="ccf83-195">NuGet package search works the same on nuget.org, from the NuGet CLI, and within the NuGet Package Manager extension in Visual Studio.</span></span> <span data-ttu-id="ccf83-196">Im Allgemeinen wird die Suche auf Schlüsselwörter und auf die Paketbeschreibungen angewendet.</span><span class="sxs-lookup"><span data-stu-id="ccf83-196">In general, search is applied to keywords as well as package descriptions.</span></span>

- <span data-ttu-id="ccf83-197">**Schlüsselwörter:** Die Suche ermittelt relevante Pakete, die eines der bereitgestellten Schlüsselwörter enthalten.</span><span class="sxs-lookup"><span data-stu-id="ccf83-197">**Keywords**: Search looks for relevant packages that contain any of the provided keywords.</span></span> <span data-ttu-id="ccf83-198">Beispiel: `modern UI`</span><span class="sxs-lookup"><span data-stu-id="ccf83-198">Example: `modern UI`.</span></span> <span data-ttu-id="ccf83-199">Verwenden Sie„+“ zwischen Begriffen wie `modern+UI`, um nach Paketen zu suchen, die alle der angegebenen Schlüsselwörter enthalten.</span><span class="sxs-lookup"><span data-stu-id="ccf83-199">To search for packages that contain all of the provided keywords, use "+" between the terms, such as `modern+UI`.</span></span>
- <span data-ttu-id="ccf83-200">**Ausdrücke:** Wenn Sie Begriffe in Anführungszeichen eingeben, ermittelt die Suche Übereinstimmungen mit diesen Begriffen, bei denen die Groß-/Kleinschreibung nicht berücksichtigt wird.</span><span class="sxs-lookup"><span data-stu-id="ccf83-200">**Phrases**: Entering terms within quotation marks looks for exact case-insensitive matches to those terms.</span></span> <span data-ttu-id="ccf83-201">Ein Beispiel: `"modern UI" package`</span><span class="sxs-lookup"><span data-stu-id="ccf83-201">Example: `"modern UI" package`</span></span>
